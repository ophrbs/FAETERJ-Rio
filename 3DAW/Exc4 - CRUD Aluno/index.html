CRUD Aluno

Orientações

Criar um CRUD de aluno e postar no Github.
Criar quatro funcionalidades, Incluir novos alunos, alterar alunos existentes, lista todos os alunos, listar um aluno e excluir aluno.
Criar um menu com cada uma das funcionalidades.
A alteração, a exclusão e a listar um aluno precisa saber qual aluno será encontrado, então deveapresentar uma tela para obter a matricula desse aluno.

Utilize a barra de navegação para acessar as funções do CRUD
